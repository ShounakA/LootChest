# LootChest
Your personal chest of loot! 

## What is it?

Loot Chest is a GraphQL Service that helps you keep track of you IRL inventory!

## Really what is it?

This is a project to learn about GraphQL and .NET 6.

## Prerequisites

- .NET 6 
## How to start

```bash
$ dotnet restore
$ dotnet run
```

Navigate to {domain}/graphql

### Uses
- .NET 6
- HotChocolate
